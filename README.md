# Config files backup for some applications

[中文说明](README_ZH.md)

1. Directory Structure:

   `{Application-Name}/{Config-Files & Update-Date-Files}`

2. Including:

   - Clash for Windows

     - images
     - cfw-settings.yaml
     - README.md

   - Git

     - .gitconfig

   - JetBrains IDE (Based on IntellJ IDEA)

     - plugins-installed.txt
     - OddyCodeStyle.xml

   - Prettier

     - .prettierrc

   - Vim

     - .ideavimrc

   - Windows Terminal

     - settings.json

   - ZSH

     - .zshrc
